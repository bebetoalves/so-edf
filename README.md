## Usage

```bash
$ npx degit bebetoalves/vite-vue-ts-minimal-template my-app

$ cd my-app

$ git init # Required if you want a repository and work with Git hooks

$ yarn install

$ yarn dev
```

## Available commands

Run in development mode

```bash
$ yarn dev
```

Create production build

```bash
$ yarn build
```

Run Vue Compiler

```bash
$ yarn type-check
```

Serve production build locally

```bash
$ yarn preview
```
